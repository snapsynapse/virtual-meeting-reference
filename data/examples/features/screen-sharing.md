## Summary

Ability to share a full desktop, specific application window, or browser tab with meeting participants in real time.

## What Counts

- Full desktop sharing
- Single application or window sharing
- Browser tab sharing with audio
- Annotation tools on shared content
- Remote control of shared screen

## What Does Not Count

- Sending screenshots as static images
- File transfer without live visual sharing
